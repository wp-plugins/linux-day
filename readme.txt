This plugin adds a widget for Linux Day banners.

== Description ==

Simply add Linux Day banners to your sidebar with a handy widget.

== Installation ==

1. Download and extract the plugin's archive.
2. Put the extracted folder in the `/wp-content/plugins/` directory.
3. Activate the plugin in the WordPress Admin.

Or simply install it from the WP Admin Plugins section.
